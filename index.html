Works

View project Foodentify Foodentify is an iOS application that recognizes the food on your plate and provides nutrition facts based on the results. It was built using Swift and  the Apple CoreML framework.
View project Janet's Bakery Janet's Bakery is an example of a freelance project. I assisted my client in designing and building an elegant and user-friendly web application to streamline order processes and display information.
View project Cryptotracker Cryptotracker is a cross-platform mobile application that tracks the fluctuations of 20 cryptocurrencies. It was built using the React Native framework.

About Me

Hi there, my name is Wayne Yu. I aspire to become a software engineer with an ambition to dream big and the skillset to realize it.

I have long been fascinated by the intersections between the worlds of technology and business. I truly believe that every technological solution can contribute to a meaningful purpose and generate a significant impact.

Aside from work, I enjoy dancing and creating video content about my life on Youtube. I also cofounded The Byte Labs, an educational Youtube channel that introduces pre-college students in Taiwan to the world of programming.
